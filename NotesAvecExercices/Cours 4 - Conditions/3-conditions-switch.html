Le switch

Le switch sert à choisir une action selon une valeur précise. C'est pratique quand on compare la même variable à plusieurs valeurs possibles.

A retenir

  • switch (valeur) : la valeur qu'on veut tester.
  • case "X" : un cas possible (comparaison stricte ===).
  • break; : on arrête le switch (très important).
  • default : cas "sinon" si rien ne correspond.

Erreur classique : oublier break

Sans break;, JavaScript continue au case suivant même si ce n'est pas le bon (on appelle ça le fall-through).
