Si age = 17, que renvoie l'expression age >= 18 ?

Pourquoi imbriquer des if ?

  • Pour tester plusieurs règles dépendantes
  • Pour remplacer systématiquement toutes les boucles
  • Pour déclarer des variables globales

Quelles affirmations sont correctes pour switch ?

  • `break` évite de tomber dans la case suivante.
  • `default` gère les cas non prévus.
  • Chaque `case` est une condition booléenne complète.
  • `switch` peut remplacer une longue suite de if/else if.
Lequel est l'équivalent ternaire de: 
if (estConnecte) {
  message = "Bienvenue";
} else {
  message = "Connexion requise";
}
  • message = estConnecte ? "Bienvenue" : "Connexion requise";
  • message = estConnecte : "Bienvenue" ? "Connexion requise";
  • message = ? estConnecte "Bienvenue" : "Connexion requise";

Ordonne une stratégie claire pour coder une règle métier conditionnelle.
