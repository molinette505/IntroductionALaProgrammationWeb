La manipulation du DOM : rendre le Web vivant

Modifier l'apparence

On peut changer l'apparence avec une classe CSS ou avec style.

1) Ajouter une classe avec classList.add()

element.classList.add("ma-classe")

2) Retirer une classe avec classList.remove()

element.classList.remove("ma-classe")

3) Modifier directement avec style

element.style.backgroundColor = "#fef3c7"
