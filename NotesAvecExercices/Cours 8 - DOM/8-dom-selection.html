Sélectionner un élément dans le DOM

Pourquoi sélectionner un élément HTML ?

Afin de pouvoir modifier un élément HTML, il faut d'abord le sélectionner.

Il existe deux méthodes principales que nous allons détailler dans cette page. :

  • document.getElementById("id") : cible directement un élément par son ID.
  • document.querySelector("sélecteur-css") : cible le premier élément qui correspond à un sélecteur CSS.

Sélection par ID avec getElementById

document.getElementById("monId")
  • Cible un élément unique par son ID.

Sélection par sélecteur CSS avec querySelector

document.querySelector("selecteur-css")
  • Utilise la même syntaxe que les sélecteurs CSS.
  • Retourne le premier élément qui correspond.
