Quelle méthode cible directement un élément unique par son ID ?

  • document.querySelectorAll()
  • document.getElementById()
  • document.getElementsByTagName()

Quelles affirmations sont correctes ?

  • innerText modifie le texte visible.
  • innerHTML peut injecter des balises HTML.
  • value sert à lire/écrire la valeur d'un input.
  • value remplace classList pour les classes CSS.

Glisse chaque effet dans la bonne propriété DOM.

  • Change le texte visible d'un élément
  • Injecte des balises HTML dans l'élément
  • Lit ou remplace la valeur d'un champ de formulaire
  • Ajoute une classe CSS à un élément
