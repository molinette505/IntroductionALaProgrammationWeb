Le DOM est...

  • Le fichier HTML brut stocké sur disque
  • La représentation en mémoire de la page par le navigateur
  • Uniquement une bibliothèque JavaScript externe

Quelles affirmations sont correctes ?

  • getElementById() cible un ID unique.
  • querySelector() utilise des sélecteurs CSS.
  • querySelector() retourne toujours une liste complète.
  • querySelectorAll() retourne une collection.

Quelle propriété permet d'injecter des balises HTML ?

  • innerText
  • innerHTML
  • value

Quelle propriété lit le contenu saisi dans un <input> ?

Quelles méthodes appartiennent à classList ?

  • add()
  • remove()
  • toggle()
  • appendChild()

Quelle option modifie la source d'une image DOM ?

  • image.src = "avatar.png"
  • image.innerText = "avatar.png"
  • image.value = "avatar.png"

Ordonne les étapes pour ajouter un élément dans la page.

  • Insérer dans le parent avec appendChild
  • Configurer son contenu/texte/classes
  • Créer le nœud avec createElement

Associe chaque API à son effet principal.

  • Cibler un élément unique par ID
  • Modifier le texte visible
  • Remplacer le contenu par du HTML
  • Ajouter dynamiquement un nœud dans la page
