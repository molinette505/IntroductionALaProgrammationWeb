La manipulation du DOM : rendre le Web vivant

Qu'est-ce que le DOM ?

Le DOM permet à JavaScript de modifier le contenu, la structure ou le style de la page en temps réel, sans recharger ou changer de page.

Sélectionner un élément

Afin de pouvoir modifier un élément HTML, il faut d'abord le sélectionner.

1) Sélection par ID avec getElementById

document.getElementById("monId")
  • Cible un élément unique par son ID.
  • Très utilisé pour les éléments importants de l'interface.

2) Sélection par sélecteur CSS avec querySelector

document.querySelector("selecteur-css")
  • Utilise la même syntaxe que les sélecteurs CSS.
  • Retourne le premier élément qui correspond.

Modifier le contenu

Cette section montre trois façons simples de modifier le contenu affiché dans la page.

innerText

Utilise innerText pour changer uniquement le texte visible d'un élément.

element.innerText = `${texte}`

innerHTML

Utilise innerHTML pour remplacer le contenu avec du HTML (texte + balises).

element.innerHTML = `${html}`

value

Utilise value pour lire ce qui est saisi dans un champ de formulaire.

let texte = elementInput.value

Modifier l'apparence

On peut changer l'apparence avec une classe CSS ou avec style.

1) Ajouter une classe avec classList.add()

element.classList.add("ma-classe")

2) Retirer une classe avec classList.remove()

element.classList.remove("ma-classe")

3) Modifier directement avec style

element.style.backgroundColor = "#fef3c7"

Modifier les attributs

Un attribut décrit une propriété d'un élément HTML (src, onclick, href, etc.).

  • setAttribute(nom, valeur) modifie un attribut.

Ex. Mettre à jour src d'une image

element.setAttribute("src", "url-image")
