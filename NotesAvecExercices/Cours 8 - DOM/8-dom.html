La manipulation du DOM : rendre le Web vivant

Qu'est-ce que le DOM ?

Le DOM permet à JavaScript de modifier le contenu, la structure ou le style de la page en temps réel, sans recharger ou changer de page.

Sélectionner un élément

Afin de pouvoir modifier un élément HTML, il faut d'abord le sélectionner.

Deux méthodes principales de sélection

Selection par ID

document.getElementById("monId")
  • Cible l'élément unique possédant cet ID.
  • C'est la méthode la plus performante.

Selection par sélecteur CSS

document.querySelector("selecteur")
  • Utilise la syntaxe CSS (ex : .ma-classe, #monId, div > p, [mon-attribut="50"]).
  • Il retourne le premier élément correspondant.

On peut stocker l'élément sélectionné dans une variable pour ensuite pouvoir le modifier (ajouter une classe, changer le texte d'un titre, etc.).

Modifier le contenu

Trois propriétés utiles pour modifier le contenu :

  • innerText : modifie uniquement le texte visible d'un élément.
  • innerHTML : modifie le contenu HTML interne (texte + balises).
  • value : lit ou change la valeur d'un champ de formulaire (input, textarea, etc.).

Modifier l'apparence

Pour le style, on combine souvent :

  • classList.add/remove/toggle pour appliquer une règle CSS réutilisable.
  • element.style pour un ajustement ponctuel (ex: transform).

Modifier les attributs

Un attribut décrit une propriété d'un élément HTML (href, src, title, data-*).

  • setAttribute(nom, valeur) modifie un attribut.
  • getAttribute(nom) lit sa valeur actuelle.

Création dynamique

La creation dynamique consiste a fabriquer des noeuds en JavaScript puis les inserer dans la page.

  • createElement() cree un element en memoire.
  • appendChild() l'ajoute comme enfant.
  • classList.add() et setAttribute() le configurent avant insertion.

Tableaux parallèles vers HTML

Une technique fréquente consiste à stocker plusieurs informations liées dans des tableaux parallèles (même index = même entité), puis à générer l'affichage HTML avec une boucle.

Exemple : noms[i], roles[i] et scores[i] décrivent le même joueur. On crée ensuite un élément DOM pour chaque index.
