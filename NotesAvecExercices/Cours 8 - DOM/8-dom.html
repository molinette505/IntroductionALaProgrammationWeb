Introduction au DOM

Qu'est-ce que le DOM ?

Quand le navigateur charge une page HTML, il transforme son contenu statique (le code HTML) en une structure que JavaScript peut lire et modifier.

Le DOM permet à JavaScript de modifier le contenu, la structure ou le style de la page en temps réel, sans recharger ou changer de page.

Comment visualiser le DOM

1. HTML

Vous codez la structure de la page dans un fichier .html avec des balises comme <h1>, <p> et <button>.

2. DOM

Le navigateur transforme ce code HTML statique en un arbre dynamique constitué de noeuds. Chaque élément devient un objet que JavaScript peut sélectionner.

3. JavaScript

Votre script peut ensuite lire, modifier, créer et réorganiser des éléments HTML sans recharger la page.

Ce que le DOM permet de faire

  • Lire le contenu des éléments : récupérer un titre, un paragraphe, un champ texte ou un bouton.
  • Modifier le contenu : changer le texte dans une balise h2 ou le contenu d'une div.
  • Changer l'apparence : ajouter une classe CSS ou modifier un style.
  • Modifier les attributs : changer l'attribut src d'une image.
  • Créer des éléments : créer des div, h2, ul ou li.
  • Réagir aux actions : brancher du code sur le clic ou le survol d'un élément.

Exemple : liste d'épicerie dynamique
