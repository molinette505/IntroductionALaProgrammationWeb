Fonctions classiques

Retour

Introduction

Une fonction est un bloc de code qu’on peut réutiliser pour faire une tâche précise.

On l’écrit une fois, puis on l’appelle quand on en a besoin.

Pourquoi utiliser des fonctions ?

  • Éviter la répétition dans le code.
  • Rendre le code plus clair en donnant un nom d'action (ex. calculerMoyenne(notes) à une suite d'opérations.
  • Faciliter la modification en modifiant le code à un seul endroit.

Image mentale

  • Une fonction reçoit parfois des entrées (paramètres).
  • Elle fait un travail.
  • Elle peut renvoyer une sortie avec return.

1) Déclarer et appeler une fonction

Il y a deux étapes : déclarer la fonction, puis l’appeler.

Syntaxe de base

function nomFonction() {
  // instructions
}
  • function sert à créer une fonction.
  • nomFonction est le nom de la fonction.
  • Le code entre { } s’exécute quand on appelle la fonction.

Appel

Pour exécuter une fonction, on écrit : nomFonction()

2) Paramètres et arguments

Les paramètres servent à personnaliser une fonction.

Vocabulaire

  • Paramètre : variable écrite dans la définition de la fonction.
  • Argument : valeur donnée lors de l’appel.

3) Retourner une valeur avec return

Une fonction peut retourner une valeur. Cette valeur peut ensuite être affichée, stockée, ou réutilisée.

À retenir

  • return renvoie un résultat.
  • Sans return, la fonction ne renvoie pas de valeur utile.

Sortie immédiate

Dès que JavaScript rencontre return, il quitte la fonction immédiatement.

  • Une fonction peut avoir plusieurs return.
  • Le premier return exécuté arrête la fonction.

4) Portée des variables

À retenir

Une variable créée dans une fonction n’existe que dans cette fonction. On dit qu’elle est locale.

Important

La variable bonus existe seulement dans la fonction ajouterBonus().

Mini récap

  • On déclare une fonction, puis on l’appelle.
  • Les paramètres permettent de personnaliser une fonction.
  • return permet de renvoyer une valeur.
  • Une variable déclarée dans une fonction est locale.
