Gérer des événements en JavaScript

Qu’est-ce qu’un événement ?

Un événement, c’est un “signal” envoyé par le navigateur quand quelque chose se produit : un clic, une touche du clavier, la souris qui bouge, etc.

  • 1) L’utilisateur fait une action
  • 2) Le navigateur détecte l’événement
  • 3) JavaScript exécute une fonction (notre réaction face à l'événement)

addEventListener("click")

La méthode recommandée : addEventListener. On dit au navigateur : “Quand l’événement arrive, exécute cette fonction.” 

element.addEventListener("click", function (event) {
  // code exécuté au clic
});

Le paramètre event et event.target

Quand un événement arrive, le navigateur donne un objet event à notre fonction. Cet objet contient des informations utiles : quelle touche, quel élément, etc.

event.target = l’élément exact qui a été cliqué.
event.currentTarget = l’élément sur lequel on a mis le listener (souvent celui qu’on veut modifier). 

element.addEventListener("click", function (event) {
  console.log(event.target);
  console.log(event.currentTarget);
});

Passer une fonction nommée

Au lieu d’écrire function (event) { ... } directement dans addEventListener, on peut créer une fonction séparée et la passer. 

function gererToucheAppuyee(event) {
  console.log(event.key);
}

element.addEventListener("keydown", gererToucheAppuyee);

Événement clavier keydown

On peut lire les touches tapées avec l’événement keydown. Important : clique d’abord dans le champ texte pour qu’il soit sélectionné (focus). 

element.addEventListener("keydown", function (event) {
  ...
});
