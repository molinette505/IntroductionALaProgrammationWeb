Gerer des evenements en JavaScript

Qu'est-ce qu'un evenement ?

Un evenement est une action detectee par le navigateur: clic, touche clavier, survol de souris, etc. JavaScript peut reagir a ces actions pour mettre a jour la page.

onclick dans le HTML

On peut declencher une fonction directement depuis un attribut HTML.

<button onclick="maFonction()">...</button>

Propriete element.onclick

On peut affecter une fonction de clic en JavaScript avec la propriete onclick.

element.onclick = function () { ... }

addEventListener("click")

La methode recommandee pour ecouter les interactions est addEventListener.

element.addEventListener("click", nomFonction)

event.target avec le survol

event.target represente l'element exact touche par la souris.

event.target.classList.add("ma-classe")

Evenement clavier keydown

On peut lire les touches tapees avec l'evenement keydown.

element.addEventListener("keydown", nomFonction)
