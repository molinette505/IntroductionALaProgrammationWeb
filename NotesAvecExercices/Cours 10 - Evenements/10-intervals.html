Intervals avec setInterval

setInterval sert à répéter une action automatiquement, à chaque X millisecondes (ms). clearInterval sert à arrêter cette répétition.

À retenir

  • Intervalle = une action qui se répète.
  • Callback = la fonction exécutée à chaque répétition.
  • ID d’intervalle = le numéro retourné par setInterval (utile pour arrêter).
  • ms = millisecondes (1000 ms = 1 seconde).

setInterval et clearInterval

Pour arrêter un intervalle, il faut conserver l’ID retourné par setInterval. 

let id = setInterval(maFonction, 1000);
// plus tard...
clearInterval(id);

Étapes (toujours les mêmes)

  • Créer une variable compteur (ex : tick).
  • Démarrer l’intervalle et garder son ID.
  • Dans la fonction : augmenter le compteur, puis vérifier une condition.
  • Si la condition est atteinte : arrêter avec clearInterval(id).

Mini récap

  • setInterval(fonction, duree) répète fonction toutes les duree millisecondes.
  • On garde l’ID retourné : let id = setInterval(...).
  • On arrête avec clearInterval(id).
