Délais avec setTimeout

setTimeout exécute une fonction après un certain délai (en millisecondes).

À retenir

  • Timeout = une action retardée (une fois).
  • Callback = la fonction exécutée après le délai.
  • Délais = temps avant que la fonction ne soit exécutée en millisecondes (1000 ms = 1 seconde).
  • setTimeout(fonction, délais) : lance fonction après délais millisecondes.

setTimeout (exécuter plus tard)

Important : le script continue d’exécuter le reste du code pendant que le timer “attend”. C’est pour ça que l’ordre des messages en console peut parfois surprendre.

Mini récap

  • setTimeout(fn, ms) exécute fnms.
  • Le code continue pendant l’attente (ce n’est pas “bloquant”).
  • Pour un compte à rebours, on utilise souvent plusieurs timeouts ou un intervalle (vu dans le cours précédent).
