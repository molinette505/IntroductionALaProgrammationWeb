Qu’est-ce qu’une string ?

Une string (chaîne de caractères) est du texte en programmation : lettres, chiffres, espaces, symboles.

À retenir

  • Une string est toujours entourée par des guillemets ou des backticks.
  • Exemples : "Bonjour", 'Hello', `Hola`

Créer du texte : guillemets simples ou doubles

Pour une string simple, on utilise des guillemets : ' ' ou " ".

Règle simple

  • Si le texte contient une apostrophe (ex. J'aime), utilise " "
  • Si le texte contient des guillemets (ex. "..."), utilise ' '

Astuce

On peut aussi “échapper” un caractère avec \, mais au début, le plus simple est de choisir l’autre type de guillemets.

Mettre des variables dans le texte : backticks et ${}

Les backticks (`) servent à créer des messages plus lisibles, surtout quand on veut insérer des variables.

Pourquoi c’est pratique ?

  • Interpolation : on insère une variable avec ${variable}
  • Multi-ligne : on peut écrire une string sur plusieurs lignes (utile pour du HTML)

Concaténation : combiner des strings avec +

La concaténation consiste à coller du texte ensemble:
"Score: " + 100 + " points;

Attention : string + nombre

  • "54" + 6: "546" (le nombre devient du texte)
  • 54 + 6: 60 (addition normale)

Longueur d’un texte : .length

La propriété .length donne le nombre de caractères dans une string. Les espaces et symboles comptent aussi.

Erreur fréquente

On ne compare pas la string directement : motDePasse > 0 ❌
On compare sa longueur : motDePasse.length > 0 ✅
