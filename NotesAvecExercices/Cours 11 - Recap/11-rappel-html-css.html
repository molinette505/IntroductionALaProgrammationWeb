HTML et CSS : les bases à retenir

HTML sert à construire la page (le contenu). CSS sert à choisir des éléments et à leur donner un style (l’apparence).

  • Ordre logique :
  • 1) j’écris le HTML
  • 2) je cible en CSS
  • 3) j’ajuste le style

1) Les éléments HTML

Un élément HTML est un “bloc” de la page : un titre, un paragraphe, une image, un lien, etc. La plupart ont une balise ouvrante et une balise fermante.

Exemple : la balise <p> crée un paragraphe.

<p>Bonjour tout le monde</p>

Éléments courants (à connaître)

  • <h1> à <h6> : titres et sous-titres.
  • <p> : paragraphe.
  • <div>, <section>, <article> : regrouper du contenu.
  • <img> : image.
  • <a> : lien.
  • <input>, <button> : interaction.

2) Les attributs HTML

Les attributs ajoutent des informations à un élément. Ils s’écrivent dans la balise ouvrante (ex : id, class, src, href).

Exemple : ici, id, class, type et value sont des attributs.

<input id="prenom" class="champ-saisie" type="text" value="Lina">

id

  • Un id identifie un seul élément (unique dans la page).
  • Très utile pour viser un élément précis.
  • On peu par exemple ajouter un style CSS sur un élément précis.
<h1 id="titre-principal">Bienvenue</h1>

class

  • Une class peut être partagée par plusieurs éléments.
  • On peu par exemple ajouter un style CSS sur plusieurs éléments qui ont la classe.
<p class="important">Attention</p>

src et alt

  • src = fichier à charger.
  • alt = description (utile si l’image ne charge pas + accessibilité).
<img src="avatar.png" alt="Portrait de Léa">

href

Pour un lien : href = url de destination. 

<a href="https://exemple.com">Visiter le site</a>

3) Les sélecteurs CSS

Un sélecteur sert à choisir quels éléments HTML vont recevoir un style. 

sélecteur {
  propriété: valeur;
}

3.1) Sélecteur par id

Un sélecteur par id commence par #. Il vise un élément unique.

HTML

<h2 id="titre-principal">Bienvenue</h2>

CSS

#titre-principal {
  color: #2563eb;
}

Règle : un id doit rester unique. Si plusieurs éléments partagent le même style, utilise une class.

3.2) Sélecteur par class

Un sélecteur par class commence par .. Il peut viser plusieurs éléments.

HTML

<p class="important">Examen lundi</p>
<p class="important">Remise mardi</p>

CSS

.important {
  color: #b91c1c;
  font-weight: 700;
}

3.3) Sélecteur par balise

Quand on écrit directement le nom d’une balise, CSS cible tous les éléments de ce type. Exemple : p { ... }

HTML

<p>Premier paragraphe</p>
<p>Deuxième paragraphe</p>

CSS

p {
  line-height: 1.6;
}

3.4) Sélecteurs combinés

On peut cibler selon l’emplacement : seulement un paragraphe dans une carte

HTML

<div class="carte">
  <p>Texte dans la carte</p>
</div>

CSS

.carte p {
  color: #475569;
}
