Cheat Sheet HTML et CSS

1) La structure HTML

Conteneurs: <div> (bloc), <span> (inline).

Contenu: <h1> a <h4>, <p>, <ul>, <li>.

Interaction: <img> avec src/alt, <a> avec href, <input>, <button>.

2) Les poignées (attributs)

Attribut Pourquoi c'est crucial en JS ?
id Cibler un seul element precis.
class Cibler un groupe d'elements.
value Lire la saisie utilisateur dans un input.
style Modifier l'apparence en direct.
onclick Declencher une fonction au clic.

3) Les selecteurs CSS

Selecteurs de base: #id, .classe, balise.

Descendance: div p (descendant), div > p (enfant direct).

HTML = ce qui est ecrit sur la page.

JS = comment on le fait bouger en utilisant les deux precedents.
