HTML et CSS : les bases a retenir

Cette page resume ce que sont les elements HTML, a quoi servent leurs attributs, puis comment CSS selectionne ces elements pour leur appliquer un style.

Ordre logique : on construit d'abord la page avec HTML, puis on utilise CSS pour choisir les elements a styliser.

1) Les elements HTML

Un element HTML represente un morceau de la page. Cela peut etre un titre, un paragraphe, une image, un lien, un champ de saisie ou un bouton.

Structure classique : une balise ouvrante, un contenu, puis une balise fermante.

<p>Bonjour tout le monde</p>

Les elements les plus courants

  • <h1> a <h6> : pour les titres et sous-titres.
  • <p> : pour un paragraphe de texte.
  • <div> et <section> : pour regrouper du contenu.
  • <img> : pour afficher une image.
  • <a> : pour creer un lien.
  • <input> et <button> : pour l'interaction.

2) Les attributs HTML

Les attributs ajoutent des informations a un element. Ils s'ecrivent dans la balise ouvrante et precisent le role ou le comportement de cet element.

Dans l'exemple suivant, id, class, type et value sont des attributs.

<input id="prenom" class="champ-saisie" type="text" value="Lina">

id

Un id identifie un element unique dans la page. On s'en sert souvent en CSS et en JavaScript pour viser un element precis.

<h1 id="titre-principal">Bienvenue</h1>

class

Une class sert a donner le meme nom a plusieurs elements qui partagent le meme style ou la meme fonction.

<p class="message-important">Attention</p>

src et alt pour une image

src indique le fichier image a charger. alt decrit l'image si elle ne s'affiche pas et aide aussi l'accessibilite.

<img src="avatar.png" alt="Portrait de Lea">

href pour un lien

href contient l'adresse vers laquelle le lien doit envoyer l'utilisateur.

<a href="https://exemple.com">Visiter le site</a>

type="text" et value pour un input

type="text" indique que le champ sert a ecrire du texte. value represente la valeur actuelle du champ.

<input type="text" value="Lina">

3) Les selecteurs CSS

Un selecteur CSS sert a choisir quels elements HTML vont recevoir un style. On ecrit d'abord le selecteur, puis les proprietes entre accolades.

selecteur {
  propriete: valeur;
}

Le selecteur par id

Un selecteur par id commence par #. Il cible un element unique de la page.

HTML

<h2 id="titre-principal">Bienvenue</h2>

CSS

#titre-principal {
  color: #2563eb;
}

Un id doit rester unique dans une page. Si plusieurs elements ont besoin du meme style, il faut plutot utiliser une class.

Le selecteur par class

Un selecteur par class commence par .. Il permet de cibler plusieurs elements qui partagent le meme nom de classe.

HTML

<p class="important">Examen lundi</p>
<p class="important">Remise mardi</p>

CSS

.important {
  color: #b91c1c;
  font-weight: 700;
}

Une class est utile quand plusieurs elements doivent partager le meme style visuel.

Le selecteur par balise

Quand on ecrit directement le nom d'une balise, CSS cible tous les elements de ce type dans la zone concernee.

HTML

<p>Premier paragraphe</p>
<p>Deuxieme paragraphe</p>

CSS

p {
  line-height: 1.6;
  color: #334155;
}

Les selecteurs combines

On peut aussi cibler un element selon son emplacement dans la structure HTML, par exemple un paragraphe a l'interieur d'une carte ou un enfant direct.

HTML

<div class="carte">
  <p>Texte dans la carte</p>
</div>

<ul class="liste">
  <li>Item 1</li>
</ul>

CSS

.carte p {
  color: #475569;
}

.liste > li {
  font-weight: 700;
}

A retenir : HTML cree les elements de la page. Les attributs donnent des informations a ces elements. CSS utilise des selecteurs comme #id, .class ou p pour leur appliquer un style.
