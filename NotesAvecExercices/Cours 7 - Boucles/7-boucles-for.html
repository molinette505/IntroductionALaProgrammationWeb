La boucle For (Pour)

Elle regroupe l'initialisation, la condition et l'incrémentation sur une seule ligne.

Utile lorsqu'on sait d'avance combien de fois on veut boucler (répéter).

for (initialisation; condition; incrémentation) { 
    // code à répéter tant que la condition est vraie
}
for (let i = 0; i <= 4; i++) { 
     // cette boucle s'exécutera de 0 à 4, soit 5 fois au total
}
  • L'initialisation :
    • À partir de quelle valeur on commence à compter?
    • On déclare et initialise une variable compteur à une valeur de départ. (ex: let i = 0)
    • On utiliser souvent i, j et k par convention
  • La condition :
    • Jusqu'à quelle valeur on boucle ?
    • Testée avant chaque tour de boucle. (ex: i < 5)
  • L'incrémentation :
    • De combien on augmente (ou diminue) la variable compteur à chaque tour ?
    • Modifie la variable de compteur à la fin de chaque tour. (ex: i++)
  • Le corps de la boucle { ... } :
    • Le code à exécuter à chaque tour.
Quel sera l'affichage dans la console ? 
for (let i = 1; i <= 4; i++) {
    console.log(i);
}
  • 1, 2, 3, 4
  • 0, 1, 2, 3, 4
  • 1, 2, 3
