La boucle for (répéter)

Une boucle for sert à répéter du code. Elle est idéale quand on sait “combien de fois” (ou “jusqu’où”) on veut compter.

La recette (à mémoriser)

  • Départ : je crée un compteur (souvent i) que j'initialise à une valeur de départ
  • Condition : tant que i est plus petit ou plus grand qu'une valeur, on continue
  • Pas : j’augmente ou je diminue le compteur
  • Corps : le code exécuté à chaque tour

Le format

for (départ; condition; pas) {
  // code répété
}

Lire la boucle comme une phrase

  • for (let i = 0; i <= 4; i++) → “Je commence à 0, je vais jusqu’à 4, j’ajoute 1 à chaque répétition.”
  • for (let i = 10; i >= 0; i--) → “Je commence à 10, je vais jusqu’à 1, j’enlève 1 à chaque répétition.”
Quel sera l'affichage dans la console ? 
for (let i = 3; i <= 8; i++) {
  console.log(i);
}

Indice : la boucle commence à 1 et s’arrête quand i dépasse 4.

  • 3, 4, 5, 6, 7, 8
  • 0, 1, 2, 3, 4, 5, 6, 7, 8
  • 3, 4, 5, 6, 7

Mini récap

  • Compteur : let i = ...
  • Limite : i <= ... ou i >= ...
  • Pas : i++ ou i--
