Afficher des messages

console.log() est le premier outil pour voir ce que fait votre programme. Il sert autant à apprendre qu'à déboguer.

Rappel : Quand le programme affiche quelque chose dans la console, cela aide à vérifier si le code s'exécute comme prévu.

Afficher un message

Le développeur utilise la console pour afficher et vérifier son code en utilisant console.log().

Où voir le résultat ?

1. Dans ces notes de cours :
Pour faciliter l'apprentissage, le résultat s'affiche directement dans la console du simulateur.

2. Dans la « vraie vie » (Navigateur) :
Normalement, le résultat est caché. Pour le voir, le développeur doit ouvrir les outils de débogage :

  • Clavier : Appuyez sur la touche F12.
  • Souris : Faites un clic droit > Inspecter.

Cherchez ensuite l'onglet Console (>_).

Combiner plusieurs valeurs

On peut afficher plusieurs textes et plusieurs nombres dans un même console.log() en les séparant avec des virgules.
