Introduction

JavaScript est le langage de programmation du Web. Contrairement au HTML (contenu) et au CSS (style), JavaScript permet de créer la logique et l'interactivité.

Note importante : Le code s'exécute ligne par ligne, de haut en bas.

Le point-virgule ;

Il marque la fin d'une instruction. C'est comme le point « . » dans une phrase. Prenez l'habitude de toujours le mettre.

Afficher des données

Le développeur utilise la console pour afficher et vérifier son code en utilisant console.log().

Cherchez ensuite l'onglet Console (>_).

Les Variables

Une variable est un espace réservé dans la mémoire de l'ordinateur.

Imaginez une boîte avec une étiquette.

Étape 1

Créer

let score;
  • L'ordinateur réserve un bloc de mémoire
  • Il lui donne le nom "score".
  • À ce stade, la boîte est vide, sa valeur est undefined.
Étape 2

Écriture

score = 10;
  • On place la valeur 10 dans la boîte.
  • Si on change la valeur plus tard, l'ancienne est écrasée et perdue à jamais.

Étape 3

Lecture

console.log(score);
  • L'ordinateur va lire le contenu de la boîte
  • Il va ensuite remplacer score dans le code par sa valeur 10
  • Pour l'ordinateur console.log(score); devient console.log(10);

Types de Données

JavaScript devine le type de données automatiquement.

  • String (Texte) : Toujours entre guillemets "Bonjour!".
  • Number (Nombre) : Entier ou à virgule 3.1416
  • Boolean (Logique) : Soit true ou false

Règles de Nommage

On utilise le camelCase : premier mot minuscule, majuscule ensuite.

Correct

  • maVariable
  • scoreJoueur
  • prixTotal

Incorrect

  • MaVariable (Pas de majuscule au début)
  • ma_variable (Pas de tiret bas)
  • 1score (Ne commence pas par un chiffre)
