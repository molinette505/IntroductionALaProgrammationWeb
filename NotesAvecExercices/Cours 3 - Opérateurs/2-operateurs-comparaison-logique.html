Opérateurs de Comparaison

Ces opérateurs comparent deux valeurs et retournent un booléen (true ou false).

Grandeur

  • > Plus grand que
  • < Plus petit que
  • >= Plus grand ou égal
  • <= Plus petit ou égal

Égalité

  • === Strictement égal
  • !== Différent de

Logique Booléenne

Permet de combiner plusieurs conditions ensemble.

L'oprérateur ET (&&).

Les deux côtés doivent être vrai.

Valeur A Op Valeur B Résultat
true && true true
true && false false
false && true false
false && false false

L'opérateur OU (||).

Un des deux côtés doit être vrai.

Valeur A Op Valeur B Résultat
true || true true
true || false true
false || true true
false || false false

L'inversion du NON (!)

Inverse le booléen. true devient false

Op Valeur A Résultat
! true false
! false true
