Les arrays

Retour

Qu’est-ce qu’un tableau (array) ?

Un array (tableau) est une liste ordonnée de valeurs. On l’utilise quand on veut mettre plusieurs éléments dans une seule variable.

À retenir

  • Un array contient des éléments dans un ordre.
  • On accède à un élément avec son index (sa position).
  • Les index commencent à 0 (premier élément = index 0).
  • .length donne le nombre d’éléments.

Le concept d’index

Chaque élément est identifié par sa position dans le tableau: son index.

En programmation, on commence à compter à partir de 0.

  • 1er élément → index 0
  • 2e élément → index 1
  • 3e élément → index 2
index let scores = [250, 200, 145, 110]
[0] 250
[1] 200
[2] 145
[3] 110

Créer un tableau

Un tableau se crée avec des crochets [], souvent dans une variable.

1) Tableau vide

let filmsFavoris = [];

2) Tableau déjà rempli

let notesEtudiants = [15, 12, 18];

La taille d’un tableau (.length)

.length retourne le nombre d’éléments dans le tableau.

Exemples rapides

  • [] → length vaut 0
  • [10, 20, 30] → length vaut 3

Accéder et modifier les éléments d’un tableau

Accéder (lire)

Pour lire un élément : tableau[index]. Exemple : scores[2] donne le 3e élément.

index let scores = [250, 200, 145, 110]
[0] 250
[1] 200
[2] 145
[3] 110

scores[2] retourne donc 145

Écrire (modifier)

Pour modifier un élément : tableau[index] = nouvelleValeur.

Premier et dernier élément

Rappels

  • Premier élément : index 0
  • Dernier élément : index tableau.length - 1
index let scores = [250, 200, 145, 110]
1er [0] 250
[1] 200
[2] 145
dernier [scores.length - 1] = [4 - 1] = [3] 110

Mini récap

  • Un array est une liste ordonnée : ["Victor", "Emma", "Béatrice", "Maxime", "Nicolas", ...]
  • Index : ça commence à 0 → tableau[0]
  • Taille : tableau.length
  • Lire : tableau[i] / Modifier : tableau[i] = nouvelle valeur
